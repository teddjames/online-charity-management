## setting up the repo
> Clone the repo onto your local machine
> Activate your virtual Environment using 
  bash: 
       pipenv install,
       pipenv shell 
> install the necessary dependancies 
  Flask==2.2.5
  Flask-SQLAlchemy==3.0.3
  Flask-JWT-Extended==4.4.0
  psycopg2-binary
  Werkzeug==2.3.7
  python-dotenv==1.0.0
  Flask-Migrate==4.0.0
>run the server using:
 bash:
      python run.py
